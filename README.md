# 哈工大-密码学原理B(选修)-2026春

该门课程是计算学部开设的选修课程，24理论学时+8实验学时，以“严格可证明安全”的密码学理论为主

## 实验

### lab1 伪随机与一次一密

- 掌握伪随机生成工具使用、一次一密加密应用
- 尝试破解多次加密

### lab2 对称加密与认证

- 实现对图片的CPA安全加密和CCA安全加密
- 使用CCA能力攻击CPA安全加密方案

### lab3 公钥加密

- 使用RSA进行非对称加解密，结合对称加密方案对图片进行加密
- 使用共模攻击解密图片

### lab4 Web PKI 与 TLS

- 建立一个部署数字证书的网站
- 抓包访问网站的流量，分析浏览器和服务器建立TLS握手的过程
